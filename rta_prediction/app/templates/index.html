Probability Prediction of Road Traffic Accidents

In this project, we built a machine learning model that predicts the probability of RTAs. To predict the probability we used historical datasets from Kaggle

Learn More

This web application is built using Flask and all the html pages are integrated into the web application.
It includes an interactive RTA prediction Map . This prediction allows users to input an origin, destination and time of travel of up to 48 hours from the current time.

It will then calculate the probability of accidents in the accident hotspots that happen to lie along the best route connecting the origin and destination, as defined by Here Map's API.

Apart from Here Map, this prediction also uses real-time weather forecast data provided by Dark Sky.
